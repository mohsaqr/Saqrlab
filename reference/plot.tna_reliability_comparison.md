# Plot Method for TNA Reliability Comparison

Plot Method for TNA Reliability Comparison

## Usage

``` r
# S3 method for class 'tna_reliability_comparison'
plot(x, metric = "Pearson", type = "scatter", ...)
```

## Arguments

- x:

  A `tna_reliability_comparison` object.

- metric:

  Character. Metric to plot. Default: "Pearson".

- type:

  Character. Plot type: "scatter", "boxplot", or "histogram". Default:
  "scatter".

- ...:

  Additional arguments (ignored).
