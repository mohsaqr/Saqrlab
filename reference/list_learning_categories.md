# Get Learning State Categories Summary

Display a summary of all available learning state categories and the
number of verbs in each.

## Usage

``` r
list_learning_categories()
```

## Value

A data frame with category names and verb counts.

## Examples

``` r
list_learning_categories()
#>           Category Count
#> 1    metacognitive    20
#> 2        cognitive    30
#> 3       behavioral    30
#> 4           social    30
#> 5     motivational    30
#> 6        affective    30
#> 7 group_regulation     9
#> 8              lms    30

#> 2          Read, Study, Analyze, Summarize, Memorize, ...

#> 5            Focus, Persist, Explore, Create, Strive, ...

#> 7    Adapt, Cohesion, Consensus, Coregulate, Discuss, ...
#> 8             View, Access, Download, Upload, Submit, ...
```
