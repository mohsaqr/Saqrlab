How Should Edges Appear at Each Time Point?

Showing 4 nodes over 5 time bins. Edges exist between some pairs at some times.

A. Vertical Connectors

Dashed vertical lines between connected nodes at each time bin. Simple, but gets noisy with many edges. Current implementation uses this.

B. Dots + Curved Arcs

Nodes become visible dots at time points where they have edges. Curved arcs bow outward to connect pairs. More expressive than straight verticals.

C. Shaded Bands

Subtle filled regions between connected node trajectories during time intervals when the edge exists. Shows connection duration, not just single time points.

D. Mini Network Snapshots

Small network graph aligned below each time bin. Shows the actual topology. Clear but takes vertical space and compresses the main plot.
