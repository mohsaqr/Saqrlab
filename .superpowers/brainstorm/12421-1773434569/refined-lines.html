Take Two: Lines That Actually Look Like Lines

Which approach reads better?

A. Subtle Variable Width

Width range 0.5–3px. Segments with stroke-linecap="round" for smooth joins. Still reads as a line, thickness is noticeable but not dominant.

B. Opacity = Degree

Uniform 2px width. Degree mapped to opacity — highly connected segments are vivid, isolated ones fade. Lines always look like lines. Nodes that lose connections literally fade away.
