Smooth Variable-Width Lines

Spline-interpolated to ~200 micro-segments. No visible joints. Width varies with degree.

Implementation: Spline-interpolate both the metric and degree to ~200 fine points, then draw micro-segments with geom_segment() + stroke-linecap="round". Joints overlap seamlessly. Alternatively, use ggforce::geom_link2() which interpolates linewidth natively between endpoints.
