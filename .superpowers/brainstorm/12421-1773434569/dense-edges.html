Dense Networks: How to Show Edges Without Chaos

10 nodes, ~30 edges per time bin. Individual edge lines won't work. Here are approaches that survive density.

A. Degree as Dot Size

No individual edges drawn. Each node gets a dot at each time point sized by its degree. You see how connected each node is without drawing any edge lines. Scales to any density.

B. Highlight Parameter

User picks a node via highlight="A". That node's trajectory is bold, its neighbors are visible, non-neighbors fade. Edges drawn only for the highlighted node. Works at any density.

C. Heatmap Strip

Adjacency heatmap aligned below the trajectory plot. Each row is an edge pair, each column a time bin, intensity = edge weight. Reads all edges at a glance without drawing a single line between nodes.

D. Combine All Three

Degree dots on trajectories + highlight parameter to focus one node + heatmap strip of that node's edges below. Layered detail: overview at a glance, detail on demand.
